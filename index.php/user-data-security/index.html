Types of Data Excluded or Maintained

When you request data exclusion, the following types of data will be excluded:

  • Access data: username and password
  • Usage data (e.g., app usage statistics, preferences)

The following types of data may be maintained for a limited period or as required by law:

  • Technical logs and records - maintained for 1 year for security and troubleshooting purposes

After these periods, the data will be permanently deleted from our systems.
